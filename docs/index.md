# DPC

this site contains my collection of DPC (double perfect clear), 6L (six-line), 8L (eight-line), 14L (fourteen-line) tetris setups and their perfect clear solutions. many of them have extra scoring opportunities, like TSMs (t-spin minis), TSSs (t-spin singles), TSDs (t-spin doubles), and tet PCs (tetris/quad perfect clears). these are useful for looping in-game. good for blitz/ultra. sites like tetris.com, tetr.io, jstris, etc.

click a setup for more info
